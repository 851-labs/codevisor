HerdMan Server hosts projects, coding-agent sessions, event streams, files, and terminals. The
HerdMan macOS app uses the same HTTP and WebSocket API documented here.

<Callout type="warning" title="Experimental API">
  The `/v1` API is public for experimentation, but its compatibility policy is not finalized yet.
  Pin the HerdMan Server release you test against and read release notes before upgrading.
</Callout>

## Default endpoint

A standalone server listens on `127.0.0.1:49361` unless configured otherwise. The Linux installer
binds to `0.0.0.0:49361`, enables token authentication, and can install a systemd service.

```bash
curl http://127.0.0.1:49361/v1/health
```
